Real Estate Search Real Shout Wordpress Plugin allows you to add a Real Estate search portal to your Wordpress blog.

== Description ==

Real Estate Search Real Shout Wordpress Plugin allows you to add a Real Estate search portal to your Wordpress blog. The search portal is search engine friendly, allows you to capture buyer leads, provides your visitors with simple and advanced search features, allows your visitors to view details about each property, allows your visitors to save listings, allows your visitors to contact you regarding listings and provides you with admin features for managing it all. No IDX or manual entry required.

Please visit http://www.realshout.com/realshout-wordpress-plugin-readme.php for full details on how to install and setup this plugin


== Frequently Asked Questions ==

= Why when I click to perform a search nothing happens? =

Make sure you setup a new page like described in step 3 in the installation instructions and then choose this page from the dropdown in the "Default Search" settings.

= Why is the Google map not displaying anything? =

Make sure you registered for an API key from http://code.google.com/apis/maps/signup.html and added that API key to Google Map API admin setting.
